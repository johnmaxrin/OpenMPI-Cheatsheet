# OpenMPI Cheatsheet for C++

### Compile MPI Programs 
`mpic++ sourcecode.cpp -o bin_name`
#

### Run MPI Programs
`mpirun -np # ./bin_name` 
#

### Initializing MPI Environment
`MPI_Init(int *argc, char ***argv)   //Parameters are Optional.`
#
